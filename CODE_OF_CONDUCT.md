# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1][cc] as its code of
conduct.

The full text — including the standards, scope, and enforcement guidelines —
is canonical at:

<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

## Reporting

To report behaviour that you believe violates this code of conduct, contact
the project maintainer privately at **martingdavidson@gmail.com**. Reports
are handled in confidence; you will receive an acknowledgement within 7 days.

For security vulnerabilities (not conduct issues), see
[SECURITY.md](SECURITY.md) for the private disclosure channel.

[cc]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
